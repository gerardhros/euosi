# Table with country codes used in the package

This table contains country names as used for the spatial extent where
sptfs have been derived from.

## Usage

``` r
osi_countries
```

## Format

A data.table with 249 rows and 3 columns:

- country_name:

  the name of the country

- country_code:

  the country code;from https://datahub.io/core/country-codes
