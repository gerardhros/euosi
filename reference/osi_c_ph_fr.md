# Calculate the pH index for France

This function evaluates the pH index in France

## Usage

``` r
osi_c_ph_fr(B_LU, B_TEXTURE_GEPPA, A_PH_WA, unitcheck = TRUE)
```

## Arguments

- B_LU:

  (character) The crop type

- B_TEXTURE_GEPPA:

  (character) The soil texture class in a particular region.

- A_PH_WA:

  (numeric) The pH measured in H2O

- unitcheck:

  (character) Option to switch off unit checks (TRUE or FALSE)

## Value

The pH index in France estimated from pH in water, the textural class
and the crop type
