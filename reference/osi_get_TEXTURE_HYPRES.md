# Estimate soil texture according to HYPRES classification

Estimate soil texture according to HYPRES classification

## Usage

``` r
osi_get_TEXTURE_HYPRES(A_CLAY_MI, A_SILT_MI, A_SAND_MI, type = "code")
```

## Arguments

- A_CLAY_MI:

  (numeric) Clay content (%)

- A_SILT_MI:

  (numeric) Silt content (%)

- A_SAND_MI:

  (numeric) Silt content (%)

- type:

  (character) return HYPRES classification names or codes (options:
  'name' or 'code')

## Value

Texture class according to the HYPRES classification system
