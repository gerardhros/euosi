# Estimate soil pH values (-)

Estimate soil pH values (-)

## Usage

``` r
osi_conv_ph(
  element,
  A_PH_KCL = NA_real_,
  A_PH_CC = NA_real_,
  A_PH_WA = NA_real_
)
```

## Arguments

- element:

  (character) the method requested to be calculated

- A_PH_KCL:

  (numeric) The soil pH determined in 1 M KCL extract

- A_PH_CC:

  (numeric) The soil pH determined in 0.01M CaCl2 extract

- A_PH_WA:

  (numeric) The soil pH determined in water
