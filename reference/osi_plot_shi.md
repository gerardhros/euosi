# This function calculates the phosphate availability.

This function calculates the phosphate availability.

## Usage

``` r
osi_plot_shi(shi, xpar, xpar_label = "soil indicator", pgroup = NULL)
```

## Arguments

- shi:

  (numeric) A soil health indicator value estimated with euosi

- xpar:

  (numeric) The soil property controlling the shi value

- xpar_label:

  (character) The label to be printed on the x-axis of the plot

- pgroup:

  (character) A grouping factor to show different responses of shi
