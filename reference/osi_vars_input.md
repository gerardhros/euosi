# A dataset with all input variables required by the euosi package

This table contains all input parameters needed to run the euosi package

## Usage

``` r
osi_vars_input
```

## Format

An object of class `data.table` (inherits from `data.frame`) with 155
rows and 7 columns.

## Details

- code:

  The input parameter's name

- parameter:

  A description of the parameter

- unit:

  The unit for the parameter

- data_type:

  Type of data, either boolean, numeric, integer, or character

- value_min:

  Minimum value for a numeric or integer parameter

- value_max:

  Maximum value for a numeric or integer parameter

- categories:

  Possible categories for categorial variables
